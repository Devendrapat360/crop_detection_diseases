#  Ai crop_detection_diseases
🌱 AI Crop Disease Detection System — Project Description
📌 Introduction

The AI Crop Disease Detection System is a full-stack web application that uses Artificial Intelligence and Deep Learning to detect diseases in crop leaves from uploaded images.
The system helps farmers and agriculture users identify plant diseases quickly and provides treatment suggestions.

The project is developed using:

React for frontend
Flask for backend API
TensorFlow for AI model training and prediction


🎯 Objective

The main objective of this project is:

To detect crop diseases automatically using AI
To reduce manual disease identification
To help farmers take quick action
To improve crop health and productivity
